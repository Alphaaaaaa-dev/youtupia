# AVERTI – Gas Detection & Fire Safety Monitoring System

AVERTI is an advanced gas detection and fire safety monitoring platform designed to strengthen industrial safety infrastructure through intelligent sensing, real-time monitoring, and data-driven insights.

---

## 🚀 Live Application

Production URL:  
https://www.miraenext.in

---

## 🛠 Tech Stack

The platform is built using modern, scalable technologies:

- Vite  
- React  
- TypeScript  
- Tailwind CSS  
- shadcn-ui  

---

## 🧠 System Architecture

AVERTI integrates a robust backend and automation workflow to ensure reliability and real-time responsiveness:

- **Database:** Supabase (secure, scalable cloud database)  
- **Automation & Integration:** Make.com (workflow automation and data handling)  

This combination enables efficient data processing, real-time updates, and seamless system communication.

## 🏗 Installation (Local Development)

Ensure Node.js is installed before proceeding.

### 1️⃣ Clone the repository

```bash
git clone <YOUR_GIT_URL>
2️⃣ Navigate into the project
cd averti
3️⃣ Install dependencies
npm install
4️⃣ Start development server
npm run dev

The application will run with hot-reload enabled.

📦 Production Build

To generate an optimized production build:

npm run build

The output will be available in the dist directory.

🌐 Deployment

The application is deployed using Vercel.

Any push to the main branch automatically triggers a new deployment pipeline, ensuring continuous integration and delivery.
